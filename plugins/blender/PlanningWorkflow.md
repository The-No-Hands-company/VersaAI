# Planning Workflow for AAA Asset Generation in VersaAI Blender Plugin

## Step-by-Step Example: Character Creation
1. **Intent Parsing**
   - User prompt: `create Lara Croft-like character`
   - VersaAI detects reference, style, and asset requirements.
2. **Dialogue & Clarification**
   - VersaAI: "Do you want realistic proportions, specific clothing, or a certain pose?"
   - User: "Yes, realistic, explorer outfit, standing pose."
3. **Task Decomposition**
   - Base mesh generation
   - Sculpting and detailing
   - Texturing and materials
   - Rigging and posing
   - Accessory creation (e.g., backpack, boots)
4. **Asset/Model Library Integration**
   - Search for or generate high-poly assets (hair, clothing, props)
   - Import and apply to character
5. **Execution & Feedback**
   - VersaAI generates each step, shows previews, and asks for feedback.
   - User can refine or approve each stage.

## Environment Generation Example
- Prompt: `generate AAA environment`
- Steps: terrain creation, prop placement, lighting setup, post-processing.

## Notes
- VersaAI uses advanced models and asset libraries for high-quality results.
- Multi-turn dialogue ensures user requirements are met.
