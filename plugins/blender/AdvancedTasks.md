# Advanced AI-Powered Tasks for VersaAI Blender Plugin

## Character Creation
- `create Lara Croft-like character` — Generates a realistic, high-poly female adventurer character.
- Dialogue: VersaAI will ask for details (style, pose, accessories, polycount) if prompt is vague.

## AAA Asset Generation Workflow
- `generate AAA environment` — Creates a detailed scene with realistic assets, lighting, and layout.
- Planning: VersaAI decomposes the task into steps (terrain, props, lighting, post-processing).

## Asset/Model Library Integration
- `import high-poly asset from library` — Searches and imports assets from local or online libraries.
- `add realistic lighting setup` — Applies advanced lighting presets to the scene.

## Multi-Turn Dialogue Example
1. User: `create Lara Croft-like character`
2. VersaAI: "Do you want realistic proportions, specific clothing, or a certain pose?"
3. User: "Yes, realistic, with explorer outfit and standing pose."
4. VersaAI: "Generating character with specified details..."

## Notes
- These tasks require advanced intent parsing, planning, and asset/model integration.
- VersaAI will guide users through clarifying questions and step-by-step workflows for best results.
