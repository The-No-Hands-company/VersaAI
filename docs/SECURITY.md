# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability in this project, please report it to the maintainers.

### How to Report

- Email: security@example.com
- Create a GitHub issue with the label "security"

We will acknowledge your report within 48 hours and provide a more detailed response within 7 days.

## Security Updates

Security updates will be released as soon as possible after a vulnerability is confirmed and fixed.

## Responsible Disclosure

We kindly ask that you give us reasonable time to fix the issue before making it public.
