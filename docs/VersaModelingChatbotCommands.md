# VersaModelingChatbot Supported Commands

This document lists the commands currently supported by the VersaModelingChatbot module.

## Commands

- **generate cube**
  - Generates a 3D cube in VersaModeling.

- **list tools**
  - Displays a list of available modeling tools: Extrude, Bevel, Sculpt, Paint.

- **help**
  - Shows a list of supported commands.

- **exit**
  - Exits VersaModeling assistant with a goodbye message.

## Example Usage
```
Enter command: help
Commands: generate cube, list tools, help, exit.
```
