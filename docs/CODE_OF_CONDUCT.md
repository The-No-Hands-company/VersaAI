# Code of Conduct

This project adheres to a code of conduct to ensure a welcoming environment for all contributors.

## Our Standards

- Be respectful and inclusive
- Use welcoming and inclusive language
- Be collaborative
- Show empathy towards other community members

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by contacting the project team.

For more details, see [Contributor Covenant](https://www.contributor-covenant.org/version/2/0/code_of_conduct/).
