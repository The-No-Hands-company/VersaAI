## 🎯 What Each File Does

### model_router.py
**Purpose:** Smart model selection based on task analysis

**Key Classes:**
- `TaskComplexity(Enum)` - 5 complexity levels
- `ModelTier(Enum)` - Fast/Balanced/Powerful
- `ModelSpec` - Model specifications dataclass
- `ModelRouter` - Main routing logic

**Key Methods:**
- `select_model()` - Selects optimal model for task
- `_detect_language()` - Auto-detects programming language
- `_detect_complexity()` - Analyzes task complexity
- `_score_model()` - Scores model fitness (0-100)
- `list_models()` - Lists all available models

**Supports:** 5 models (Phi-2, DeepSeek, StarCoder2, CodeLlama, WizardCoder)

---

### model_ensemble.py
**Purpose:** Coordinates multiple models for optimal results

**Key Classes:**
- `ModelEnsemble` - Multi-model coordinator

**Key Methods:**
- `generate()` - Main generation method with mode selection
- `_generate_auto()` - Router-based selection
- `_generate_fast()` - Fastest model
- `_generate_quality()` - Best quality model
- `_generate_consensus()` - Multi-model voting
- `get_stats()` - Performance statistics
- `unload_model()` / `unload_all()` - Memory management

**Generation Modes:**
1. **auto** - Router decides (recommended)
2. **fast** - Use Phi-2 (quick responses)
3. **quality** - Use CodeLlama/WizardCoder (best output)
4. **consensus** - Use 3 models, combine results

## 🚀 Usage Examples

### Basic Usage
```python
from versaai.models.model_router import ModelRouter
from versaai.models.model_ensemble import ModelEnsemble

# Initialize
router = ModelRouter(available_ram_gb=16)
ensemble = ModelEnsemble(router, model_loader_function)

# Generate code (auto mode)
result = ensemble.generate(
    task="Create a Python binary search function",
    mode="auto"
)

print(result['code'])
print(f"Generated by: {result['model_name']}")
```

### Consensus Mode (Critical Code)
```python
result = ensemble.generate(
    task="Implement user authentication with JWT",
    mode="consensus"
)

# Review all model outputs
for output in result['consensus']['all_results']:
    print(f"\n{output['model']}:")
    print(output['code'])
```

### Manual Model Selection
```python
# Let router analyze and select
model_id, spec = router.select_model(
    task="Debug memory leak in C++",
    language="c++"
)

print(f"Recommended: {spec.name}")
# Output: WizardCoder-15B (specializes in debugging)
```

## 🔄 Integration Points

### 1. With Code Model

```python
# In versaai/models/code_model.py
from .model_router import ModelRouter
from .model_ensemble import ModelEnsemble

class CodeModel:
    def __init__(self):
        self.router = ModelRouter()
        self.ensemble = ModelEnsemble(self.router, load_llama_model)
    
    def generate_code(self, task, context, **kwargs):
        return self.ensemble.generate(
            task=task,
            language=context.language,
            mode=kwargs.get('mode', 'auto')
        )
```

### 3. Real LLM Loading
```python
# Model loader function
from llama_cpp import Llama
from pathlib import Path

def load_llama_model(model_id):
    models_dir = Path.home() / ".versaai" / "models"
    
    model_files = {
        "phi2": "phi-2.Q4_K_M.gguf",
        "deepseek": "deepseek-coder-6.7b-instruct.Q4_K_M.gguf",
        "starcoder2": "starcoder2-7b.Q4_K_M.gguf",
        "codellama": "codellama-13b-instruct.Q4_K_M.gguf",
        "wizardcoder": "wizardcoder-15b-v1.0.Q4_K_M.gguf"
    }
    
    return Llama(
        model_path=str(models_dir / model_files[model_id]),
        n_ctx=8192,
        n_gpu_layers=32,  # GPU offload
        verbose=False
    )
```

## 🎓 Key Concepts

### Model Selection Algorithm
1. Analyze task complexity (simple/medium/complex/debug/refactor)
2. Detect programming language
3. Score each model (0-100 points)
4. Consider user preferences (speed vs quality)
5. Apply resource constraints (RAM available)
6. Select highest scoring model

### Consensus Mode
1. Select 3 diverse models (different tiers)
2. Run all in parallel (ThreadPoolExecutor)
3. Compare results for similarity
4. Return best result + all alternatives
5. User can review all outputs

### Resource Awareness
- 4GB RAM → Phi-2 only
- 8GB RAM → Phi-2, DeepSeek, StarCoder2
- 16GB+ RAM → All models available
